================================================================
키움증권 OpenAPI+ 설치 가이드 (간단 버전)
Phoenix Trading System 실행을 위한 필수 설치
================================================================

📥 1단계: 다운로드

공식 다운로드 링크:
https://www.kiwoom.com/h/customer/download/VOpenApiInfoView

위 링크 접속 → "KOAStudio 다운로드" 버튼 클릭
파일명: KOAStudioSetup.exe (약 30MB)

================================================================

🔧 2단계: 설치

1. KOAStudioSetup.exe 우클릭
2. "관리자 권한으로 실행" 선택 ⭐중요!
3. "예" 클릭 (UAC 창)
4. 설치 마법사 따라가기:
   - 사용권 계약 동의
   - 설치 경로: 기본값 사용 (C:\OpenAPI)
   - 구성 요소: 전체 설치 권장
   - 설치 진행 (1~3분)
5. 설치 완료 → 컴퓨터 재시작 (권장)

================================================================

✅ 3단계: 설치 확인

방법 1: 제어판 확인
- 제어판 → 프로그램 및 기능
- "Kiwoom Open API+" 항목이 있으면 성공 ✅

방법 2: KOAStudio 실행
- 시작 메뉴 → "KOAStudio" 검색
- 프로그램이 실행되면 성공 ✅

방법 3: 레지스트리 확인 (고급)
- Win+R → regedit 입력
- HKEY_CLASSES_ROOT\KHOPENAPI.KHOpenAPICtrl.1 존재 확인

================================================================

🔐 4단계: OpenAPI 사용 권한 (선택)

1. 영웅문 HTS 다운로드 및 설치
   https://www.kiwoom.com/h/customer/download/O_Down

2. 영웅문 로그인

3. 상단 메뉴 → 설정 → OpenAPI 관리
   → "신청" 버튼 클릭 → 즉시 승인

================================================================

⚠️ 문제 해결

문제 1: "관리자 권한이 필요합니다"
해결: 설치 파일 우클릭 → "관리자 권한으로 실행"

문제 2: "설치 중 오류"
해결:
- 기존 버전 제거 (제어판 → 프로그램 및 기능)
- 컴퓨터 재시작
- 백신 프로그램 일시 중지
- 관리자 권한으로 재설치

문제 3: "COM 등록 실패"
해결:
- 명령 프롬프트(관리자) 열기
- cd C:\OpenAPI
- regsvr32 KHOpenAPI.ocx
- 성공 메시지 확인

문제 4: Python에서 import 실패
해결:
- 32비트 Python 사용 확인 (필수!)
- OpenAPI 재설치
- 레지스트리 등록 확인

================================================================

💡 중요 사항

✅ 키움증권 계좌 필요
✅ 미국 주식 거래 권한 필요 (SOXL용)
✅ Windows 전용 (Mac, Linux 불가)
✅ 32비트/64비트 Windows 모두 가능
✅ 관리자 권한 설치 필수
✅ 설치 후 컴퓨터 재시작 권장

❌ 64비트 Python은 작동 안 함
   → Phoenix는 32비트 Python으로 빌드해야 함
   → 하지만 사용자 PC는 64비트 Windows 가능
